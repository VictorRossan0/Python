Importação Metabase Financeiro

Procedimento

Para garantir uma importação perfeita no Banco de Dados do Metabase Financeiro, siga as etapas abaixo:

  1. Verifique o arquivo fornecido por Kelvin para garantir que não exista nenhum 'IdOper' diferente dos valores existentes na tabela operacoes (1 ao 31).
  2. Após verificar e remover esses 'IdOper' do arquivo, modifique a coluna Competencia para o formato AAAA-MM-DD(Exemplo: 2024-01-01).
  3. Prossiga com a conversão do arquivo.
  4. Selecione o arquivo CSV e converta-o para o formato Excel.
  5. Selecione o arquivo Excel baixado automaticamente e ajuste as colunas conforme necessário.
  6. Importe o arquivo Excel baixado automaticamente para o Banco de Dados.
